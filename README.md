[![Supabase](https://img.shields.io/badge/Supabase-Powered-3ECF8E)](https://supabase.com)

**منصة متكاملة لإدارة العيادات الطبية بالذكاء الاصطناعي**

Clinic AI Pro هي حل إداري متطور يجمع بين قوة الذكاء الاصطناعي (Claude AI) ونظام ERP مبسط، مصمم خصيصاً للعيادات الطبية والمستشفيات الصغيرة والمتوسطة.

---

## ✨ المميزات الرئيسية

### 🎯 لوحة تحكم مركزية
- **بطاقات إحصائية ذكية** تعرض أهم المؤشرات (الأطباء، المرضى، الإيرادات، الفواتير)
- **تنبيهات فورية** للمخزون المنخفض، الفواتير غير المسددة، الرواتب المعلقة، الأدوية منتهية الصلاحية
- **أزرار إجراءات سريعة** للوصول المباشر إلى الوظائف الرئيسية
- **مساعد ذكي مصغر** للإجابة على أسئلة سريعة عن العيادة

### 🤖 المساعد الذكي (Claude AI)
- محادثة تفاعلية كاملة مع فهم سياق بيانات العيادة
- إجابات دقيقة مبنية على أرقام حقيقية من قاعدة البيانات
- أسئلة مقترحة لتوجيه المستخدم
- حفظ تاريخ المحادثة ضمن الجلسة

### 📈 تحليلات متقدمة
- رسوم بيانية تفاعلية (Plotly) للإيرادات والمواعيد
- تحليل الخدمات الأكثر طلباً وربحية
- توزيع الفواتير والمصروفات حسب الفئات
- مخططات مقارنة لأداء الأطباء

### 🔮 التنبؤ والتخطيط
- توقع الإيرادات للأيام القادمة (7-90 يوم)
- نطاقات ثقة إحصائية
- حساب نسبة النمو المتوقعة
- تحليل الاتجاهات المستقبلية

### 👨‍⚕️ إدارة الأطباء
- تقييم أداء الأطباء بالإيرادات والمواعيد
- مخطط شعاعي لمقارنة الأداء
- نظام الرواتب والمكافآت والخصومات
- متابعة الحضور والانصراف

### 🏥 إدارة المرضى
- قاعدة بيانات متكاملة للمرضى
- تحليل ديموغرافي (العمر، الجنس، الحالة)
- تقييمات المرضى وآرائهم
- ربط المرضى بالأطباء المعالجين

### 💊 إدارة المخزون والأدوية
- متابعة الكميات والحدود الدنيا
- تنبيهات انتهاء الصلاحية (قبل 90 يوم)
- تصنيف الأدوية حسب الفئات العلاجية
- تقارير الأدوية الأكثر استهلاكاً

### 📋 الوصفات الطبية
- إدارة الوصفات والبنود الدوائية
- تحليل أنماط الوصف والأدوية الأكثر شيوعاً
- تعليمات الجرعات ومدة العلاج

### 💰 نظام الفواتير (ERP)
- إصدار فواتير متكاملة مع تفاصيل البنود
- تسديد الفواتير (كامل/جزئي)
- تحميل الفواتير كنص قابل للطباعة
- إدارة المصروفات حسب الفئات
- تحليل مالي شامل (Waterfall Chart)

### 📅 الحضور والرواتب
- تسجيل الحضور والانصراف والتأخير
- حساب ساعات العمل
- إدارة الرواتب الشهرية مع المكافآت والخصومات
- تقارير حالة صرف الرواتب

### 🔔 نظام الإشعارات
- تنبيهات للأحداث الهامة (مواعيد، مدفوعات، أدوية)
- تصنيف حسب النوع والمستلم
- حالة القراءة (مقروء/غير مقروء)

### 🧠 التقارير الذكية
- 5 أنواع من التقارير الشاملة (مالي، أداء، موارد بشرية، رعاية مرضى، استراتيجي)
- دعم اللغتين العربية والإنجليزية
- إمكانية تحميل التقرير كنص

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **Streamlit** | واجهة المستخدم وتطبيق الويب |
| **Supabase** | قاعدة البيانات (PostgreSQL) وAPI |
| **Claude AI (Anthropic)** | الذكاء الاصطناعي والتحليلات |
| **HuggingFace** | نسخة احتياطية مجانية للذكاء الاصطناعي |
| **Plotly** | الرسوم البيانية التفاعلية |
| **Pandas** | معالجة وتحليل البيانات |
| **NumPy** | العمليات الحسابية والإحصائية |
